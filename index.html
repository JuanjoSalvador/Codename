Codename

Codename is another music library written in Java, as my final course project. Currently is in Alpha phase, it needs more developing and developers interested in this project.

It uses a Java Swing Interface, but if you don't like Swingyou can download it with GTK interface, as you like!

A music player

Codename supports MP3, FLAC and OGG audio files. Can play all of this types, and soon, much more! Also it can plays files in your computer and online music files (like audio streamings).

A music organizer

Codename can read MP3 metadata and organize your music in folders, like the path Artist/Album/Track. Currently only MP3 is supported on the organizer, but soon, other types are available.
